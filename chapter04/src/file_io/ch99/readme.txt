[ 실행 흐름 ]

1.	다어어리에 글을 작성할 수 있는 패널클래스에서 작성후 버튼을 누르면
	콜백 인터페이스의 메서드를 호출하여 버튼을 눌렀다고 콜백한다.
	( 패널 클래스에서 글을 작성한 날짜를 보기위해 Calendar 클래스로 날짜를 정의한 클래스를 생성하여
	날짜가 자동으로 찍히게 구현하였음) 
   
   
2.	콜백받은 프레임은 인터페이스를 구현하여 파일에 출력하라고 정의해두었다.


[ 클래스 ]

CallBackCheckButton.java
	버튼에 이벤트가 일어나면 콜백하기 위한 인터페이스
	
CheckDate.java
	작성일의 날짜를 표현하기 위해 Calendar클래스를 가져와 구현함.

WritePanel.java
	사용자가 다이어리를 작성할 수 있도록 TextArea와 콜백 메서드 호출하기 위한 JButton이 생성되어 있고,
	CallBackCheckButton 인터페이스의 메소드를 호출한다.
	
WritFrame.java
	사용자가 작성할 패널을 프레임에 올리고, 
	콜백이 일어나면 CallBackCheckButton 인터페이스를 구현한 메소드에서 파일을 저장한다.
